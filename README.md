# SophaxPlay

Minimalist music player for MP3 and FLAC files.
Built with Python, PySide6, and pygame.

## Features

- MP3 and FLAC playback
- Album grouping (from ID3 / Vorbis tags)
- Album art display
- Playlist with double-click to play
- Play / Pause / Stop / Next / Previous
- Volume control
- Progress scrubber with seek

## Install

```bash
pip install pygame mutagen PySide6
```

## Run

```bash
python sophaxplay.py
```

## Requirements

- Python 3.9+
- pygame ≥ 2.1.0
- mutagen ≥ 1.46.0
- PySide6 ≥ 6.4.0
